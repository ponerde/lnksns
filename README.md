LiteAdmin
===============

> 运行环境要求PHP7.2+，兼容PHP8.1

![](https://gitee.com/DengJe/LiteAdmin/raw/master/public/static/images/readme/1.png)

![](https://gitee.com/DengJe/LiteAdmin/raw/master/public/static/images/readme/3.png)

## 安装

~~~
composer install
~~~

## 前端（进入web目录）
~~~
npm install
~~~
